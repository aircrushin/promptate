# Promptate

a helpful web app for prompt generating

## Vue 3 + TypeScript + Vite + Flask + OpenAI

## Recommended Global Packages

- [Node.js]
- [pnpm]
- [Python]

## Run Client

```bash
pnpm install
pnpm run dev
```

## Run Server

```bash
cd server
pip install -r requirements.txt
python app.py
```
